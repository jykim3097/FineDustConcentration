# FineDustConcentration
### 미세 먼지 농도를 알아보자

# 한 해 중 미세먼지 농도가 높을 때는 언제인가?

### 수행 동기 (210525)
* 공기청정기를 구매하고 싶은데, 단순히 봄에만 미세먼지가 많으면 장기적인 관점에서 필요 없겠다는 생각이 들었다.
* 그래서 한 해에 미세먼지 농도 정보를 알고 싶다는 생각이 들었는데,
* 쉽게 찾아볼 수 있는 정보지만, 직접 크롤링을 통해 알아보고 싶었다.

## 크롤링을 위한 준비
